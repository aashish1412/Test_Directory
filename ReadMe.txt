To run the app on windows:

1. Go to the path where you have taken the checkout of the Code. Open cmd and install flask ('pip install flask').

2. Install virtual environment ('py -m pip install --user virtualenv').

3. Create a virtual environment ('py -m venv env').

4. Activate virtual environment ('.\env\Scripts\activate').

5. Run command- ('set FLASK_APP=app.py').

6. Run command- ('flask run') -> This will run flask on 'https://127.0.0.1:5000/'

7. Now run the Angular project 'MyApp' on 'localhost:4200'.


To run the app on Linux:


1.Go to the path where you have taken the checkout of the Code. Open cmd and install flask ('pip install flask').

2.Install virtual environment ('python3 -m pip install --user virtualenv').

3. Create a virtual environment ('python3 -m venv env').

4.Activate virtual environment ('source env/bin/activate').

5.Run command- ('export FLASK_APP=app.py').

6.Run command- ('flask run') -> This will run flask on 'https://127.0.0.1:5000/'

7.Now run the Angular project 'MyApp' in any IDE(Visual Studio,Pycharm etc) on 'localhost:4200'.
